JPG, PNG, and PDF supported — a PDF's pages are split into individual images automatically. Order matters: 1st = front, 2nd = back, 3rd = front...
Each completed pair (front + back) becomes one merged output.
For JPEG/PNG output, one file downloads per pair. For PDF, all pairs combine into one multi-page file.
Dropping a PDF splits it into one image per page, which then fills the next available slot(s) just like a JPG/PNG would.
"Max file size" applies to JPEG only (compresses to fit). "Max dimensions" resizes the image and works for both JPEG and PNG.
